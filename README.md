# Excel Sales Dashboard Project

## Project Overview
This project contains an interactive Excel dashboard built to analyze sales performance, KPIs, and business insights.

## Key Features
- Sales trend analysis
- Category-wise performance breakdown
- KPI tracking
- Interactive filters and slicers
- Data cleaning and validation

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- VLOOKUP / XLOOKUP
- Data Validation

## Insights Generated
- Identified top-performing categories
- Highlighted revenue trends
- Improved reporting visibility through dashboard visualization
